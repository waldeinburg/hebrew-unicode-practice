Windows and Linux users: It's highly recommended to install SBL Hebrew or Ezra SIL.

You can give the input field focus by clicking, but you must use the arrow keys to place the cursor (not working in Internet Explorer). And you can't complete deletion assignments by selecting text – you must use Delete or Backspace.

Notice that the cursor will be placed at the end of the text when starting.

Tip: You can quickly shift between Hebrew and your ordinary keyboard layout with keyboard shortcuts (see the keyboard settings for your operating system).

About cursor movement

Many programs use logical cursor movement as default, in which the cursor follows the logical order of the characters: Left arrow moves towards the beginning of the text and right arrow moves towards the end of the text.
When writing in mixed direction, as in the Roman/Hebrew case, this is contrary to visual cursor movement, in which the arrow keys moves left/right on the screen "as you'd expect".

Some programs let you choose, but not all, e.g. Logos. The point of these exercises are mostly to experiece the quirks of writing mixed Roman and Hebrew characters.

Not all the following information about various programs is verified:

Internet Explorer

Only logical cursor movement supported.

Safari and Google Chrome

Only visual cursor movement supported. Not recommended for these exercises.

Firefox

Visual cursor movement is default, but you can switch to logical this way:

  1. Enter the following address: about:config
  2. Find bidi.edit.caret_movement_style (use the filter).
  3. Double-click and set the value to 0 instead of 2.
Also set bidi.edit.delete_immediately to true to equate the behavior of text editors.

Microsoft Word

Logical cursor movement is default, but a Cursor movement option is found somewhere in Advanced options.
Notice that this does not work when Shift is hold down (for selecting text).

OpenOffice

Logical cursor movement is default, but you can switch to visual cursor movement this way:

  1. Select Tools → Options → Language Settings → Languages
  2. Select Enable for complex text layout (CTL) and press OK
  3. Select Tools → Options → Language Settings → Complex Text Layout
  4. Set Cursor control to Visual.
Notice that this does not work when Shift is hold down (for selecting text).
